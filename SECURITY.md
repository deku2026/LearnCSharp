# Security Policy

This is a personal C# / .NET learning scratchpad — no production runtime, no
network surface, no user data. The shipped code is a collection of
`int Run(string[])` placeholders dispatched by reflection from `Program.cs`.

If you think you found a real security issue (e.g. a malicious dependency
update slipping through Dependabot, or a CI workflow weakness), please file
a private report rather than a public issue so it can be triaged before
disclosure.
